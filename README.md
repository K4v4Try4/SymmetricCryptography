# Symmetric Key Cryptography (SPN)

This project implements **symmetric key cryptography** using a **Substitution-Permutation Network (SPN)**.

---

## Description

Symmetric key cryptography uses **the same secret key** for both encryption and decryption.  
- The sender encrypts plaintext with the secret key to produce **ciphertext**.  
- The receiver uses the same key to recover the **original message**.

---

## SPN Architecture

The algorithm is based on a **Substitution-Permutation Network**:  
- **Substitution (S-box):** modifies the bits of the data.  
- **Permutation (P-box):** rearranges the position of bits.  

These steps are repeated in multiple **rounds** to increase security, mixing and complicating the plaintext.
